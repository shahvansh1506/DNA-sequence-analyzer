# DNA Sequence Analyzer 🧬

A simple Python-based bioinformatics tool that analyzes a DNA sequence from a FASTA file.

## Features
- Calculates sequence length
- Computes GC count and GC percentage
- Computes AT count and AT percentage

## Technologies Used
- Python
- Git
- GitHub

## How to Run

1. Clone the repository:
```
git clone https://github.com/shahvansh1506/DNA-sequence-analyzer.git
```

2. Navigate into the folder:
```
cd DNA-sequence-analyzer
```

3. Run the script:
```
python main.py
```
